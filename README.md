Android的增强版`android.widget.ProgressBar`，带数字的水平滚动条（支持长方形、平行四边形及椭圆角矩形），用法和`ProgressBar`一样。

## 开发环境

- JDK 1.8
- Android Studio 4.1.3
- Android SDK 30
- Gradle 6.5
- Plugin@FindViewByME 1.4.3
- Plugin@GsonFormatPlus 1.5.8
- 刘海屏模拟器(Pixel 3 XL API 30)
